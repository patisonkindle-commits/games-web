Pattern Movement

Cat-and-mouse pattern movement demos — from basic step sequences to complex precomputed path matrices on tile grids.
Each demo explores a different approach to defining and traversing movement patterns.

6 Pattern Variants
🌀

Basic Sequence Step

Simple 4-step square pattern — Forward, Left, Back, Right (50px each). Mouse follows a hard-coded ControlData sequence on an open canvas. The simplest form of pattern movement.

🔀

Continuous ZigZag

Arrow moves through a 5-segment zigzag pattern with rotation & speed control. Arrow keys to steer, ↑↓ for speed. Y-bounds bug fixed from original C# code.

⌨️ Interactive 🐛 Fixed →
🧩

Tiled Basic Grid

Cat walks a rectangle path on a 20×20 tile grid using Bresenham step interpolation. 4 absolute-position segments define the path. Loop animates continuously.

🗺️ 20×20 🐱 Cat →
📏

Normalization Delta

Cat follows a rectangle path using relative delta segments — each stores the offset from the previous endpoint. Bresenham stepping with accumulated sum for position reconstruction.

➕ Delta 🔄 Loop →
🌐

Complex 8-Seg Path

Cat navigates an 8-segment complex path using relative delta + Bresenham interpolation. Transitions: 4→11→24→27→24→17→13→5→2. Demonstrates non-rectangular multi-segment patterns.

🧵 8 segments 🔄 Loop →
🎯

Pat Matrix Precalc

The most advanced variant. Pre-computes the entire 7-segment pattern onto a 20×20 matrix. Cat uses random-walk on marked tiles while avoiding backtracking. Toggle between 4/8 direction modes.
